# Bonethug

Project Skeleton Manager

## Installation

Add this line to your application's Gemfile:

    gem 'bonethug'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bonethug

## Usage

### Commands

	$ bonethug install {rails3|silverstripe3|drupal|php|sinatra}
